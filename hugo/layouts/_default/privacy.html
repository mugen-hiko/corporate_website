{{ .Site.Params.corporate.name.ja }} (以下 当社) は、個人情報の重要性を認識し、個人情報を保護することが社会的責務であると考え、個人情報に関する法令等を遵守し、以下に定める当社の個人情報保護方針に従い、個人情報の取得、利用、管理において適切な保護に努めます。

{{ partial `h.html` (dict `size` 2 `title` `適用範囲`) }}

本方針は、当社が行う各種サービスにおいて、お取り引き事業者様や当社お問い合わせフォームなどからご提供いただいた個人情報もしくはそれに準ずる情報を取り扱う際に、当社が遵守する方針を示したものです。

{{ partial `h.html` (dict `size` 2 `title` `個人情報の利用目的`) }}

当社では、取得した個人情報を以下の目的で利用します。

  1. 当社のサービス、事業に関するお問い合わせの回答
  2. 当社のサービス、イベント、アンケートのご案内の送付
  3. 当社のサービス、事業に関する統計資料の作成
{{ partial `h.html` (dict `size` 2 `title` `個人情報の安全管理`) }}

当社は、個人情報の安全管理について、行政、公的機関の定めるガイドラインに基づき、個人データの取り扱いに係る規律の整備、物理的および技術的な安全管理措置などの対策を講じます。

{{ partial `h.html` (dict `size` 2 `title` `個人情報の第三者提供`) }}

当社は、本人の同意を得ることなく第三者への個人情報の提供は行いません。
ただし、以下の場合は、本人の同意なしに個人情報を提供することがあります。

  1. 法令に基づく場合
  2. 人の生命、身体、財産の保護に必要であって、本人の同意取得が困難な場合
  3. 公衆衛生、児童の健全育成に必要であって、本人の同意取得が困難な場合
{{ partial `h.html` (dict `size` 2 `title` `個人情報の取り扱い業務の委託`) }}

当社は、利用目的の達成に必要な範囲内において、他の事業者へ個人情報の取り扱い業務を委託することがあります。
この場合、当社は委託契約において、委託先に対しても本方針に沿った管理、監督を行います。

{{ partial `h.html` (dict `size` 2 `title` `個人情報の開示、訂正、削除など`) }}

開示、訂正、削除などのお申し出があった場合には、ご本人様であることを確認のうえ対応します。
